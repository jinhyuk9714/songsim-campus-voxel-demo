# Campus notes

Use these as semantic anchors for the Songsim Campus scene.

## Core interpretation

- Main gate near the lower edge
- Campus rises uphill toward the academic ridge
- Mid-campus green / 하늘동산 acts as a visual breathing space
- Upper-right zone contains the sports ground, pharmacy building, and concert hall cluster
